How to Play

GOAL

Eat as many food pellets as you can. Each one scores 10 points. Don't hit the walls or your own tail.

COMMANDS

Type a command in the console and press RUN or Enter to move the snake:

snake.move("right", tiles=2)

Valid directions: right, left, up, down. tiles=N is optional; omit it to move 1 tile.

TURN TIMER

You have 1 minute to submit each command. If the timer reaches zero before you run a valid command, the session ends — SESSION TIMED OUT.

ERRORS

Invalid syntax shows a console error. You can correct and retry within the remaining turn time.

CHALLENGE MODE

Food spawns are seeded — same code means same positions for everyone. Share your code for a fair async head-to-head.
